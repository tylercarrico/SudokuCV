# SudokuCV
Sudoku puzzle solver that can read puzzle from image using computer vision. Implemented using Pyhton and OpenCV. 

# Dependencies:
* Python 2.7
* Numpy
* OpenCV

# Instructions:
## run 
* python sudoku.py image_path.jpg
